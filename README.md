## 💡 Como uso este repositório

Cada exercício resolvido recebe um commit individual com uma mensagem descritiva. Os comentários no código explicam o raciocínio, não apenas a solução.

```python
# Exemplo de como documento meu código:

# Solicita o nome do usuário e armazena na variável
nome = str(input("Qual é o seu nome? "))

# Cumprimenta o usuário usando f-string
print(f"Olá, {nome}! Seja bem-vindo.")
```
